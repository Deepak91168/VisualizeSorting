## Installation

- Fork the Repository and clone it at the desired location

```bash
  git clone https://github.com/<UserName>/VisualizeSorting.git
```
- Open Repository
```bash 
   cd VisualizeSorting
```
- Open `index.html` with your browser or run using liveServer.
